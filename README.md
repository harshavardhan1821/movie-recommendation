# Movie Recommendation System

Simple Streamlit app that provides movie recommendations using TF-IDF, K-Means clustering, and cosine similarity.

## Features

- Choose a movie from the dataset and get 5 similar recommendations.
- Visualize movie clusters using PCA + Plotly.

## Prerequisites

- Python 3.8+ (your environment)
- Recommended to use a virtual environment

## Install dependencies

Open a terminal in the project root and run:

```powershell
python -m pip install --upgrade pip
python -m pip install streamlit pandas numpy scikit-learn plotly
```

If you'd like, create a `requirements.txt` later for reproducibility.

## Run the app

From the project root run:

```powershell
cd "d:\movie recommendation"
streamlit run "New folder\app.py"
```

Then open `http://localhost:8501` in your browser.

## Files

- `New folder/app.py` - Streamlit application
- `movies.csv` - Dataset used by the app
